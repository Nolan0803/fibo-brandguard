# FIBO BrandGuard Assets

This directory contains the visual assets for FIBO BrandGuard:

## Logo Files
- `logo.png` - Main FIBO BrandGuard logo
- `logo-dark.png` - Dark theme version (if needed)
- `favicon.ico` - Browser favicon (if needed)

## Usage
The logo files should be placed in this directory. The application will automatically detect and display them.

To add the logo:
1. Save your logo image as `logo.png` in this assets folder
2. The application will automatically display it in the header
3. If no logo is found, it falls back to a professional text header

## Logo Requirements
- Format: PNG (preferred) or JPG
- Recommended width: 400-600 pixels
- Transparent background preferred for best integration
